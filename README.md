# videoteca-digital-python-5
Sistema de videoteca digital desarrollado en Python para la Fase 5 de Fundamentos de Programación.

Sistema desarrollado en Python para la gestión y consulta de películas utilizando matrices, estructuras repetitivas y funciones.

Funcionalidades
Visualización del catálogo completo
Búsqueda de películas por género
Búsqueda por calificación mínima
Consulta de películas populares y recientes
Generación de estadísticas
Validaciones iterativas de datos
Tecnologías utilizadas
Python
